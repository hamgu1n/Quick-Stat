## Section 5.3 – Counting Techniques

Computing probabilities using the formula $P(A) = \frac{\text{favorable outcomes}}{\text{total outcomes}}$ requires us to count accurately. For simple problems — like drawing a single card or rolling one die — counting by hand is easy. But what about the number of ways to arrange a 10-person committee, or the number of possible lottery tickets? This is where systematic counting techniques become essential.

### The Multiplication Rule of Counting

The **fundamental counting principle** (also called the multiplication rule) states: if one event can occur in $m$ ways and a second independent event can occur in $n$ ways, then together they can occur in $m \times n$ ways. This extends to any number of stages.

$$
\text{Total outcomes} = n_1 \times n_2 \times \cdots \times n_k
$$

For example, if a restaurant offers 4 appetizers, 6 main courses, and 3 desserts, the number of possible three-course meals is $4 \times 6 \times 3 = 72$.

### Permutations

A **permutation** is an arrangement of objects where *order matters*. If you are choosing 3 students from a class of 20 to be president, vice-president, and secretary, the order matters — student A as president and student B as vice-president is different from student B as president and student A as vice-president.

The number of permutations of $n$ objects taken $r$ at a time is:

$$
P(n, r) = \frac{n!}{(n-r)!}
$$

where $n!$ (read "n factorial") means $n \times (n-1) \times (n-2) \times \cdots \times 1$.

### Combinations

A **combination** is a selection of objects where *order does not matter*. If you are choosing 3 students from a class of 20 to form a committee (no distinct roles), then {A, B, C} and {C, B, A} are the same committee — they count as one.

The number of combinations of $n$ objects taken $r$ at a time is:

$$
C(n, r) = \binom{n}{r} = \frac{n!}{r!(n-r)!}
$$

The key distinction: use permutations when the arrangement or assignment of roles matters; use combinations when you are just selecting a group with no regard for order.

### Key Points – Counting Techniques
- The multiplication rule: if event 1 has $n_1$ outcomes and event 2 has $n_2$ outcomes, together they have $n_1 \times n_2$ outcomes
- Permutations count ordered arrangements: $P(n, r) = n! / (n-r)!$ — use when order matters
- Combinations count unordered selections: $C(n, r) = n! / (r!(n-r)!)$ — use when order does not matter
- Every permutation problem can be converted to a combination problem by dividing out the number of orderings: $P(n, r) = C(n, r) \times r!$
- The factorial $n! = n \times (n-1) \times \cdots \times 1$; by convention, $0! = 1$
- Counting techniques connect directly to probability: once we can count outcomes, we can compute probabilities for equally likely sample spaces
