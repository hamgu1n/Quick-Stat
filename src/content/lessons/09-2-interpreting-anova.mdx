## Section 9.2 – Interpreting ANOVA Results

When an ANOVA test yields a significant result (we reject $H_0$), we know that at least one group mean differs from the others. But we don't yet know *which* groups differ, or *how much* they differ. This section addresses how to move from the overall ANOVA result to specific, actionable conclusions — through post-hoc testing and careful interpretation of results.

### The Problem of Multiple Comparisons

After rejecting the ANOVA null hypothesis, it might be tempting to run pairwise t-tests for each pair of groups. As mentioned at the start of this chapter, this approach inflates the overall Type I error rate. With 3 groups there are 3 pairwise comparisons; with 5 groups there are 10. Running each at $\alpha = 0.05$ makes false positives increasingly likely as the number of comparisons grows.

**Post-hoc tests** are specifically designed to make multiple comparisons while controlling the overall error rate. Two of the most common are:

- **Tukey's Honestly Significant Difference (HSD)**: compares all possible pairs of group means while holding the family-wise error rate at $\alpha$. It is the most commonly used post-hoc test when group sizes are equal or similar.
- **Bonferroni correction**: divides the significance level by the number of comparisons, making each individual test harder to pass. It is more conservative than Tukey's HSD.

### Reading the ANOVA Output

Statistical software typically presents ANOVA results in a table with columns for the source of variation (Between/Within), sum of squares (SS), degrees of freedom (df), mean square (MS), the F-statistic, and the p-value. Learning to read this table efficiently is an essential skill.

A significant p-value in the ANOVA table tells us the overall model is significant — that the group variable explains a meaningful portion of the variation in the response. From there, post-hoc output shows which specific pairs of means are significantly different, usually with confidence intervals for the difference in means.

<div className="example">
**Example 9.2.3 – Effect Size in ANOVA**

Statistical significance tells us whether an effect exists, but not how large it is. A common measure of effect size for ANOVA is **eta-squared** ($\eta^2$):

$$
\eta^2 = \frac{\text{SSB}}{\text{SST}}
$$

where SST = SSB + SSW is the total sum of squares.

Using the values from Example 9.2.2 (SSB = 84.4, SST = 437.8), calculate $\eta^2$ and interpret it. General guidelines: $\eta^2 \approx 0.01$ is small, $\approx 0.06$ is medium, $\approx 0.14$ is large.
</div>

### Key Points – Interpreting ANOVA Results
- A significant ANOVA F-test means at least one group mean differs; it does not identify which groups
- Running multiple pairwise t-tests after ANOVA inflates the Type I error rate — use post-hoc tests instead
- Tukey's HSD is the most common post-hoc test; it controls the family-wise error rate while comparing all pairs
- The Bonferroni correction divides $\alpha$ by the number of comparisons — more conservative than Tukey
- Effect size ($\eta^2$ = SSB / SST) measures practical significance beyond statistical significance
- Always report post-hoc results alongside the overall ANOVA so readers can see which specific groups differ
